Metodologia Scrum

Desenvolvimento de um site didático e interativo com foco na Metodologia Ágil Scrum, voltado especialmente para estudantes, professores e iniciantes na área de desenvolvimento de software. O projeto tem como principal objetivo explicar de forma clara e acessível os fundamentos do Scrum, incluindo seus papéis (Product Owner, Scrum Master e Time de Desenvolvimento), cerimônias (Sprint Planning, Daily Scrum, Sprint Review e Retrospective) e artefatos (Product Backlog, Sprint Backlog e Incremento).

HTML5, CSS3, JavaScript, PostgreSQL

Github Repository

Ignis

O Projeto Ignis é uma plataforma web desenvolvida com o objetivo de monitorar e apresentar informações ambientais relacionadas a incêndios florestais em todo o território brasileiro. A aplicação permite ao usuário consultar, de forma interativa, áreas com risco de fogo, registros históricos de áreas queimadas e focos de calor ativos, utilizando dados espaciais georreferenciados.

HTML5, CSS3, JavaScript, Typescript, React, PostgreSQL

Github Repository

Aetheris

O Projeto Aetheris é uma aplicação web full stack, seu propósito é facilitar a visualização, consulta e comparação de dados geo-espaciais de satélites gratuitos. Aetheris se posiciona como um catálogo inteligente e interativo de dados de satélite, voltado principalmente a pesquisadores, estudantes e profissionais que trabalham com sensoriamento remoto e agricultura. Ele reduz o esforço de “garimpar” dados em múltiplas fontes, oferecendo em uma única interface: mapa, metadados organizados, filtros avançados, séries temporais e ferramentas de comparação visual.

HTML5, CSS3, JavaScript, MongoDB
